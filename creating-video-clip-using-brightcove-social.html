Brightcove Social permet de créer des clips vidéo à partir de vidéos existantes sur votre compte {{site.product_short}}. Plusieurs raisons peuvent vous inciter à créer un clip:

  • Pour créer un court clip d'accroche pouvant être publié sur les réseaux sociaux
  • Pour limiter la longueur d'une vidéo que vous souhaitez publier (la plupart des plateformes limitent la longueur des vidéos)
  • Vous avez constaté que les vidéos plus courtes sont plus performantes sur les réseaux sociaux

Création d'un clip

Une fois qu'un clip est enregistré, il sera enregistré en tant que vidéo sur votre compte {{site.product_short}}. Suivez ces étapes pour créer un clip.
  1. Ouvrez le module Social.
  2. Cliquez sur Vidéos dans le menu de navigation de gauche. Une liste des vidéos de votre compte {{site.product_short}} s'affichera.
  3. Cliquez sur la vidéo à partir de laquelle créer un clip, puis cliquez sur Clip. L'éditeur de clips s'ouvre.
    • Pour créer un clip, la vidéo source doit avoir au moins un rendu MP4 disponible. Si le bouton Clip est désactivé, il est possible que le rendu MP4 de la vidéo ne soit pas disponible.
    • Vous ne pouvez pas créer un clip à partir d'un autre clip. Si vous avez sélectionné une vidéo découpée, le bouton Clip sera désactivé.
    • Vous ne pouvez pas créer un clip à partir d'une vidéo dont la gestion des droits numériques est activée (le verrou apparaît dans la miniature de la vidéo) et qui a été transcodée à l'aide d'un profil d'acquisition hérité. Les vidéos compatibles DRM transcodées à l'aide d'un profil d'acquisition Dynamic Delivery peuvent être découpées. Si un clip ne peut pas être créé, le bouton Clip sera désactivé.
    • Les vidéos avec des sous-titres codés peuvent être découpées via Social, mais la vidéo découpée qui en résulte ne comportera pas de sous-titres. Vous devrez télécharger de nouvelles légendes dans la vidéo coupée, et Social les utilisera lors de la publication.
  4. Saisissez un nom de clip.
  5. Faites glisser les poignées qui apparaissent sur la chronologie pour découper le début et la fin de votre vidéo. Vous pouvez également saisir des heures de début et de fin spécifiques à l'aide des champs d'édition. Un contrôle plus précis lors de l'édition peut être obtenu à l'aide des raccourcis clavier. Lorsque vous avez terminé, cliquez sur Continuer.
    Remarque: Le lecteur dans la fenêtre modale d'écrêtage passera à une image ou deux de l'image réelle à l'heure affichée. Le comportement est le même, que vous utilisiez la souris pour faire glisser les curseurs ou que vous utilisiez les raccourcis clavier pour définir les points de découpe. L'éditeur de clips effectue les clips exactement aux heures indiquées.
  6. Passez en revue les informations du clip. Le nom du clip, les balises et tous les champs personnalisés créés pour le compte peuvent être modifiés.
    Remarque: Vous pouvez supprimer les balises qui feront apparaître le clip sur vos listes de lecteur existantes ou sa synchronisation automatique avec une destination sociale.
  7. Lorsque vous avez terminé, sélectionnez une option:
    • Enregistrer: démarre le transcodage, puis enregistre le clip dans votre compte {{site.product_short}}.
    • Enregistrer et publier... - Démarre le transcodage, puis enregistre le clip dans votre compte {{site.product_short}}. Vous pouvez également sélectionner la ou les destinations vers lesquelles publier la vidéo. Il peut y avoir un délai pour que les clips publiés apparaissent sur vos sites de réseaux sociaux car la vidéo doit être transcodée.
    • Annuler: supprime le clip.

Les clips enregistrés sont transcodés à l'aide du profil d'acquisition désigné comme découpage social par défaut pour le compte. Pour plus d'informations sur la définition des profils d'acquisition par défaut, consultez la section Gestion des profils d'ingestion.

Notez que lors de la création de clips, la vidéo source entière doit être téléchargée. Pour les vidéos sources plus longues, il peut s'écouler plus longtemps avant que le clip ne soit disponible sur votre compte {{site.product_short}}.

Remarque: Le nom du fichier source du clip sera défini sur le même nom de fichier source que la vidéo source.
