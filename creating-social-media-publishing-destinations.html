Une destination de publication est un compte/une page de réseau social sur lequel vous souhaitez publier {{site.product_short}} vidéos. Avant de pouvoir publier des vidéos sur des comptes de réseaux sociaux à l'aide de Brightcove Social, vous devez créer au moins une destination. Les destinations peuvent être créées pour publier des vidéos sur YouTube, Facebook, Twitter et LinkedIn.

Création d'une destination YouTube

Créez une destination YouTube lorsque vous souhaitez publier des vidéos sur une chaîne YouTube. Avant de créer une destination YouTube, vous pouvez vérifier votre compte YouTube. Une fois que vous aurez vérifié votre compte, vous pourrez mettre en ligne des vidéos de plus de 15 minutes.

Remarque: Si la chaîne YouTube est masquée, la destination ne fonctionnera pas. Vous devrez réactiver la chaîne YouTube pour voir les vidéos mises en ligne et reconnecter la destination {{site.product_short}} pour envoyer des vidéos.

Pour créer une destination YouTube, procédez comme suit.

  1. Ouvrez le module Social.
  2. Cliquez sur Destinations dans le menu de navigation de gauche.
  3. Cliquez sur Ajouter une destination.
  4. Cliquez sur Se connecter avec YouTube.
  5. Vous serez invité à vous connecter à votre compte Google (si ce n'est pas déjà fait). Si vous avez plusieurs comptes, assurez-vous de vous connecter au compte associé à votre chaîne YouTube.
  6. Une boîte de dialogue s'ouvre pour demander l'autorisation de gérer votre compte YouTube. Vérifiez en haut à droite de la boîte de dialogue pour confirmer que vous utilisez le bon compte Google. Si ce n'est pas le cas, utilisez le menu déroulant pour sélectionner le compte Google sur lequel vous souhaitez diffuser des vidéos. Cliquez sur Autoriser.
  7. Donnez à la destination un nom descriptif unique. Cliquez sur Ajouter.
  8. Les informations de base et les paramètres de destination par défaut s'affichent. Cliquez sur le bouton Modifier dans la section Informations de base pour modifier le nom de la destination. Consultez la section Définition des valeurs par défaut de YouTube pour plus d'informations sur la configuration des valeurs par défaut.
  9. Cliquez sur Destinations dans la marge de gauche et vérifiez que la destination apparaît dans la liste des destinations. La valeur située sous le nom de la destination est l'ID de destination.

Cliquez sur le nom de la destination pour modifier la destination. Cliquez sur le lien dans la colonne Channel/Page pour ouvrir la chaîne YouTube dans un autre onglet du navigateur.

Définition des paramètres par défaut de YouTube

Les valeurs par défaut de destination seront utilisées comme propriétés vidéo par défaut lors de la publication de vidéos sur les plateformes sociales. Pour configurer les paramètres de publication par défaut pour la destination YouTube, cliquez sur le nom de la destination, puis cliquez sur Modifier dans la section Valeurs par défaut de cette destination. Les propriétés vidéo suivantes peuvent être configurées:

  • Infos sur le poste
  • Horaire
  • Légendes
  • Visibilité
  • Licence
  • En direct

Configuration des propriétés Post Info

Cliquez sur le lien Post Info. Les propriétés vidéo YouTube suivantes peuvent être définies:

  • Description: options permettant d'utiliser la description courte ou longue {{site.product_short}} ou de saisir une description par défaut à utiliser
  • Code de langue d'origine - Utilisé pour ajouter des titres et des descriptions de vidéos traduits aux vidéos ; Entrez un code de langue à 2 lettres et sélectionnez-le dans la liste
  • Catégorie YouTube: catégorie de contenu de la vidéo
  • Tags: option permettant d'inclure ou d'exclure les balises {{site.product_short}}
  • Inclure les métadonnées {{site.product_short}} en tant que balises YouTube - Option permettant d'inclure l'identifiant de la vidéo et l'ID de référence en tant que balises YouTube
  • Tags supplémentaires - Possibilité de saisir des balises supplémentaires pour les vidéos
  • Source de la miniature: options permettant d'utiliser la miniature YouTube, la miniature {{site.product_short}} ou de télécharger votre propre image miniature

Configuration des propriétés de planification

Cliquez sur le lien Planifier et sélectionnez un paramètre de planification:

  • Publier immédiatement - Pour publier la vidéo immédiatement
  • Utiliser la date de début de Video Cloud: permet d'utiliser la date de début de Video Cloud définie pour la vidéo. Dans ce cas, nous demandons à YouTube de rendre la vidéo publique à compter de la date de début prévue-fr/ spécifiée dans Video Cloud. Si cette date est passée, YouTube rend la vidéo publique immédiatement.
Remarque: Pour que la visibilité de la vidéo soit définie sur Non répertoriée ou Privée, définissez l'option Planifier sur Publiez immédiatement et définissez l'option Visibilité sur Non répertorié ou Privé.

Configuration des propriétés Captions

Cliquez sur le lien Légendes et sélectionnez une option:

  • Utiliser les sous-titres de Video Cloud - Le module Social tentera de distribuer les sous-titres associés à la vidéo sur YouTube. Le module Social tentera d'accéder aux fichiers de sous-titres où qu'ils se trouvent. Si une URL distante est utilisée pour spécifier l'emplacement des fichiers de sous-titres et que les fichiers sont accessibles (par exemple, sans restriction géographique de telle sorte que nous ne puissions pas y accéder), Social les distribuera sur YouTube.
  • Aucune légende - Aucune légende n'est utilisée
Remarque: Pour plus d'informations sur l'utilisation des sous-titres dans Video Cloud, voir Présentation: Ajout de sous-titres à une vidéo.

Configuration des propriétés de visibilité

Cliquez sur le lien Visibilité et sélectionnez un paramètre de confidentialité pour la vidéo. Les paramètres de confidentialité suivants sont disponibles:

  • Public - Tout le monde peut visionner la vidéo
  • Non listé - Toute personne disposant du lien vers la vidéo peut la visionner
  • Privé - La vidéo ne peut être vue que par vous et les utilisateurs que vous sélectionnez
  • Use YT_Privacy_Status Custom Field - Attribue dynamiquement le paramètre de confidentialité à l'aide d'un champ personnalisé associé à la vidéo. Pour plus d'informations, consultez la section Définition de l'état de confidentialité d'une vidéo.

Il existe également un paramètre permettant d'avertir les abonnés lorsque la vidéo est publiée sur YouTube. Si cette case est cochée, vos abonnés verront vos vidéos dans leur flux Abonnements et peuvent recevoir une notification les informant que la vidéo a été mise en ligne.

Remarque: Pour plus d'informations sur les paramètres de confidentialité de YouTube, consultez la documentation YouTube.

Configuration des propriétés de licence

Cliquez sur le lien Licence et sélectionnez un paramètre Licence et droits de propriété pour la vidéo. Les paramètres suivants sont disponibles:

  • Licence YouTube standard
  • Creative Commons - Attribution
Remarque: Pour plus d'informations sur la licence YouTube et la propriété des droits, consultez la documentation YouTube.

Configuration des propriétés Live

Cliquez sur le lien Live et sélectionnez le paramètre DVR pour la vidéo. Les paramètres suivants sont disponibles:

  • Activer
  • Désactiver

Création d'une destination Facebook

Créez une destination Facebook lorsque vous souhaitez publier des vidéos sur une page Facebook. Notez que les vidéos ne peuvent être distribuées que sur des pages Facebook, et non sur des pages de profil personnelles. Pour créer une destination Facebook, procédez comme suit.

  1. Ouvrez le module Social.
  2. Cliquez sur Destinations dans le menu de navigation de gauche.
  3. Cliquez sur Ajouter une destination.
  4. Cliquez sur Se connecter avec Facebook.
  5. Une boîte de dialogue s'ouvre pour vous demander de vous connecter à votre compte Facebook. Saisissez vos informations d'identification et cliquez sur Se connecter.
  6. Une boîte de dialogue s'ouvre pour confirmer ce que Brightcove Social est autorisé à faire. Modifiez les paramètres selon vos besoins, puis cliquez sur Terminé.
  7. Une boîte de dialogue s'ouvre pour confirmer que Brightcove Social est lié à Facebook. Cliquez sur OK.
  8. Donnez à la destination un nom descriptif unique. Si vous avez plusieurs pages Facebook, sélectionnez la page sur laquelle publier à l'aide de la liste déroulante, puis cliquez sur Ajouter.
    Remarque: Vous ne pouvez pas publier de vidéos sur des pages de profil personnelles.
  9. Les informations de base et les paramètres de destination par défaut s'affichent. Cliquez sur Modifier dans la section Informations de base pour modifier le nom de la destination. Consultez la section Définition des valeurs par défaut de Facebook pour plus d'informations sur la configuration des valeurs par défaut.
  10. Cliquez sur Destinations dans la marge de gauche et vérifiez que la destination apparaît dans la liste des destinations. La valeur située sous le nom de la destination est l'ID de destination.

Cliquez sur le nom de la destination pour modifier la destination. Cliquez sur le lien dans la colonne Canal/Page pour ouvrir la page Facebook dans un autre onglet du navigateur.

Suppression d'une destination

Pour supprimer une destination, procédez comme suit:

  1. Ouvrez le module Social.
  2. Cliquez sur Destinations dans le menu de navigation de gauche. Une liste de destinations s'affiche.
  3. Cliquez sur le nom de la destination à supprimer.
  4. Les détails de la destination s'affichent. Cliquez sur Supprimer, puis sur Supprimer pour confirmer la suppression.
Remarque: Le bouton Supprimer n'apparaît pas si des tâches de synchronisation automatique créées utilisent la destination. Vous devez supprimer les tâches de synchronisation automatique avant de supprimer la destination.

Se reconnecter à une destination

Brightcove Social affichera un message d'erreur si l'une des destinations rencontre des problèmes.

Une icône d'erreur () s'affiche en regard du nom de la destination. Cela peut se produire si les autorisations Brightcove sont supprimées du compte de réseau social ou lorsque les informations d'identification utilisées lors de la création de la destination ne sont plus valides. Pour résoudre le problème:

  1. Cliquez sur le nom de la destination.
  2. Cliquez sur le lien Reconnecter en haut de la page.
  3. Vous serez peut-être invité à vous réauthentifier auprès du compte de réseau social.
