Pour télécharger des vidéos plus longues sur YouTube, vous devez vérifier votre compte auprès de YouTube:

  1. Ouvrez la page de mise en ligne YouTube à l'adresse https://www.youtube.com/upload.
  2. Cliquez sur le lien Augmenter votre limite en bas de la page.
    Remarque: Si vous ne trouvez pas le lien Augmentez votre limite, il est possible que vous soyez déjà en mesure de mettre en ligne de longues vidéos. Consultez la section Vidéos plus longues de la page Fonctionnalités de votre compte YouTube pour vérifier si la fonctionnalité est déjà activée.
  3. Suivez les étapes pour vérifier votre compte à l'aide d'un téléphone mobile.
  4. Lorsque vous recevez un SMS de YouTube en réponse, saisissez le code de vérification contenu dans le message texte et cliquez sur Soumettre.

YouTube vous confirmera alors que votre compte YouTube a été activé pour les mises en ligne plus longues. À partir de ce moment, toutes les vidéos de plus de 15 minutes de votre compte {{site.product_short}} seront synchronisées avec YouTube.
