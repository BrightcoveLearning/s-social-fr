Brightcove Social fournit plusieurs rapports qui peuvent être utilisés pour surveiller les performances de vos vidéos publiées sur vos comptes de réseaux sociaux:

  • Tableau de bord - Fournit un aperçu de haut niveau des performances vidéo sur les comptes de réseaux sociaux. Pour plus d'informations, consultez la section Examen de Social Analytics.
  • Vues sociales: fournit une vue d'ensemble de la vue sociale des vidéos partagées sur des comptes de réseaux sociaux. Couvert dans cette rubrique.
  • Engagement social - Fournit un aperçu de l'engagement social (J'aime, partages, retweets, etc.) pour les vidéos partagées sur des comptes de réseaux sociaux. Pour plus d'informations, voir Révision de Social Engagement Analytics.

Pour accéder à Social View Analytics, ouvrez le module Social, cliquez sur Social Analytics dans le menu de navigation de gauche, puis cliquez sur l'onglet Social Views en haut de la page.

Commencez par sélectionner une période pour le rapport. Pour afficher le rapport d'analyse Social Views, cliquez sur l'onglet Social Views en haut de la page.

Le rapport d'analyse de Social View est organisé selon les sections suivantes:

  • Points de vue sociaux
  • Détails sur les vues sociales

Points de vue sociaux

Le graphique Vues sur les réseaux sociaux affiche le nombre de vues des vidéos publiées sur vos comptes de réseaux sociaux pendant la période sélectionnée. La date de mise à jour est la dernière date pour laquelle {{site.product_short}} dispose de données d'analyse sociale. En général, il y a un délai de plus de 24 à 48 heures avant que les plateformes sociales ne mettent à jour les vues et les données d'engagement dans leurs API.

Remarque: Les API Twitter Insights ne signalent pas les vues des médias. Par conséquent, les données de visionnage ne sont pas disponibles dans le cadre des analyses sociales.

La légende à droite du graphique peut être utilisée pour sélectionner le niveau de détail à afficher:

  • Mes sites et applications: affiche le nombre de visionnages de vidéos des lecteurs et applications Brightcove
  • Facebook - Afficher le nombre de vidéos vues sur Facebook
  • YouTube - Afficher le nombre de visionnages de vidéos sur YouTube
  • LinkedIn - Afficher le nombre de visionnages de vidéos sur LinkedIn
  • Total: affiche le nombre total de visionnages de vidéos sur toutes les plateformes

Le survol d'une ligne du graphique affichera les données de cette date.

Détails sur les vues sociales

La section Détails des vues sociales peut être utilisée pour afficher les détails de la vue vidéo en:

  • Vues (par défaut)
  • Durée moyenne de visionnage
  • % moyen regardé

Une fois la sélection effectuée, le rapport affiche les données détaillées de chaque vidéo.

Remarque: Les API Twitter Insights ne signalent pas les vues des médias. Par conséquent, les données de visionnage ne sont pas disponibles dans le cadre des analyses sociales.

Les vues YouTube affichées dans les rapports d'analyse sont collectées à partir du rapport YouTube Analytics Overview. YouTube ne garantit pas que ces données sont des données en temps réel. Par conséquent, il se peut que vous constatiez des retards de plusieurs jours avant que les vues et l'engagement n'apparaissent dans ce rapport. Une fois que les données se trouvent dans YouTube Analytics, elles sont récupérées par les collecteurs de données nocturnes pour Brightcove Social Analytics.

Si vous avez besoin de données de visionnage en temps réel plus récentes, YouTube fournit également des rapports en temps réel avec des estimations des données de visionnage au cours des 48 dernières heures pour vos 25 vidéos les plus récemment publiées. Vous trouverez le rapport en temps réel de votre chaîne YouTube ici. Brightcove Social Analytics ne collecte pas ces données en temps réel car elles fournissent des estimations de vues (par opposition aux vues réelles) et ne s'appliquent qu'aux vidéos récentes.

Le bas du rapport comporte des commandes qui vous permettent de sélectionner le nombre de lignes à afficher et de naviguer vers d'autres pages. Cliquez sur le lien de l'un des titres de vidéos pour afficher les statistiques de vues sociales de la vidéo sélectionnée.

Exporter des données

Les données du rapport peuvent être téléchargées sous forme de valeurs séparées par des virgules et de formats Excel. Notez que la fonctionnalité d'exportation enregistre les données directement à partir d'un appel d'API Analytics. Les données exportées ne sont pas manipulées de la même manière que les données de l'interface utilisateur et contiennent actuellement toutes les données sous-jacentes du rapport, et pas seulement le sous-ensemble de champs affiché par l'interface utilisateur. Le fait de disposer de toutes les données dans le format d'origine de l'API vous permet de manipuler les données d'une manière autre que celle fournie dans l'interface utilisateur.

Questions fréquentes

1. Lorsque le module Social est utilisé pour publier une vidéo sur YouTube et que cette vidéo est ensuite intégrée à un lecteur YouTube sur un autre site, les données de lecteur/engagement seront-elles toujours collectées et transmises aux analyses du module Social ?

Social Analytics recueille toutes les données de vue signalées par YouTube, telles qu'elles apparaissent dans Creator Studio et/ou les pages d'analyse de YouTube. Si les vues du lecteur intégré YouTube sont prises en compte dans vos analyses, le module Social les saisira et les signalera. L'utilisation du code intégré de la fonction Partager > Intégrer de YouTube exactement comme indiqué fait en sorte que les vues de ce lecteur intégré soient prises en compte dans vos analyses dans YouTube. Cependant, la modification du code intégré peut empêcher YouTube de collecter les données de vues. En particulier, le fait de configurer le lecteur intégré pour lire automatiquement la vidéo semble empêcher YouTube de comptabiliser les vues de ce lecteur.
