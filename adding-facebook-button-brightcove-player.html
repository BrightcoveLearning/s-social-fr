Facebook offre la possibilité d'ajouter un bouton J'aime à côté ou en dessous d'un lecteur . Cliquez sur le bouton J'aime pour publier un lien vers la page Web hébergeant le lecteur sur la page Facebook du spectateur et ajoute une personne au nombre total de personnes qui ont aimé la vidéo. Le bouton J'aime comprend des options permettant de signaler le nombre total de likes, ainsi que d'afficher les noms et les photos de profil des amis du réseau d'un utilisateur de Facebook qui ont aimé une vidéo.

Création d'un bouton J'aime sur Facebook

Pour créer un bouton J'aime sur Facebook, procédez comme suit:

  1. Accédez au site Facebook Developers: http://developers.facebook.com/docs/reference/plugins/like/.
  2. Entrez vos préférences de bouton J'aime dans le formulaire de configuration conformément aux instructions de la page. Dans la zone URL vers J'aime, saisissez l'URL de la page hébergeant votre lecteur.
  3. Cliquez sur Obtenir le code.
  4. Cliquez sur iFrame en haut de la boîte de dialogue. Copiez le code.
  5. Ouvrez la page Web hébergeant le lecteur et collez le code sous le code de publication du lecteur. Ajoutez un tag <br/> au-dessus du code Facebook pour afficher le bouton J'aime sous le lecteur.

Lorsque la page est publiée, les photos de profil de tous les amis du spectateur qui ont aimé la vidéo seront affichées. La collection d'images ici sera différente pour chaque utilisateur de Facebook. Pour implémenter cette option, cochez Afficher les visages des amis lors de la configuration des préférences du bouton J'aime.

Cette solution fonctionne au niveau de la page Web, pas au niveau de la vidéo, et ne prend en charge qu'un seul bouton J'aime par page. Les utilisateurs de Facebook aiment le lecteur. Si un lecteur contient une liste de lecteur, la mention J'aime ne spécifie pas de vidéo individuelle. Pour publier le statut d'une vidéo individuelle, vos utilisateurs peuvent partager la vidéo à l'aide de la fonctionnalité de partage sur les réseaux sociaux.
